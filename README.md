# Subtle Weapon Rework

Public documentation for **Subtle Weapon Rework**, a Terraria mod that gives small reworks to underused vanilla weapons.

## Weapon Reworks

- [Pre-Hardmode Weapons](weapons/PreHardmode/README.md)
- [Hardmode Weapons](weapons/Hardmode/README.md)

Each weapon has its own page with its icon, current draft description, and notes.

## Author Note

This mod is created by a Japanese modder.

The English documentation was prepared with the help of translation tools, so some wording may be imperfect or unnatural.

If you find unclear descriptions, corrections and feedback are welcome.
